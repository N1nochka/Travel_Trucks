# TravelTrucks 🚐

Веб-додаток для компанії **TravelTrucks**, яка займається орендою кемперів.

Проєкт містить:
- домашню сторінку з банером;
- каталог кемперів;
- фільтрацію транспортних засобів;
- сторінку деталей кемпера;
- відгуки користувачів;
- форму бронювання.


---

# 📌 Функціонал

## 🏠 Home Page

Головна сторінка містить:
- рекламний банер;
- основний заклик до дії;
- кнопку **"View Now"**, яка переводить користувача на сторінку каталогу.

---

## 🚐 Catalog Page

Каталог містить список доступних кемперів.

Реалізовано:

- відображення карток кемперів;
- кнопку **"Show more"** для переходу на сторінку деталей;
- кнопку **"Load More"** для завантаження додаткових карток;
- додавання кемперів до обраних;
- збереження обраних у `localStorage`.

---

# 🔎 Фільтрація

Користувач може фільтрувати кемпери за:

### Локація
- текстове поле пошуку по локації.

### Тип кузова
Можна вибрати один тип:

- Alcove
- Panel Van
- Integrated
- Semi Integrated

### Двигун

- Diesel
- Petrol
- Hybrid
- Electric

### Трансмісія

- Automatic
- Manual

### Додаткове обладнання

Доступна фільтрація за характеристиками:

- AC (кондиціонер)
- Kitchen (кухня)
- Bathroom
- TV
- Radio
- Refrigerator
- Microwave
- Gas
- Water

При виконанні нового пошуку попередні результати очищуються для отримання актуальних даних.

---

# 🚐 Camper Details Page

Сторінка окремого кемпера містить:

## Галерея

- фотографії кемпера;
- перегляд зображень.

## Опис

Інформація про транспортний засіб.

## Vehicle Details

Відображаються характеристики:

- Transmission
- Engine
- AC
- Bathroom
- Kitchen
- TV
- Radio
- Refrigerator
- Microwave
- Gas
- Water

## Vehicle Information

Детальна інформація:

- Form
- Length
- Width
- Height
- Tank
- Consumption

# ⭐ Відгуки

На сторінці кемпера відображаються:

- відгуки користувачів;
- рейтинг за п'ятизірковою шкалою.

---

# 📝 Форма бронювання

Користувач може забронювати кемпер через форму.

Реалізовано:

- перевірка імені;
- перевірка email;
- валідація полів;
- повідомлення після успішного відправлення форми.

# ⚙️ Встановлення та запуск

## Клонування репозиторію

```bash
git clone <repository-link>

cd TravelTrucks
```

## Встановлення залежностей

```bash
npm install
```

## Запуск проєкту

```bash
npm run dev
```

Проєкт буде доступний:

```text
http://localhost:5173
```

---

# 📦 Production build

Створення production збірки:

```bash
npm run build
```

Перегляд production версії:

```bash
npm run preview
```
